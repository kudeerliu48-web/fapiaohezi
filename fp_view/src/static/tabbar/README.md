# TabBar 图标说明

请在此目录下放置以下 TabBar 图标文件：

## 需要的图标

1. **首页图标**
   - `home.png` (81x81px) - 默认状态
   - `home-active.png` (81x81px) - 选中状态

2. **历史图标**
   - `history.png` (81x81px) - 默认状态
   - `history-active.png` (81x81px) - 选中状态

3. **我的图标**
   - `mine.png` (81x81px) - 默认状态
   - `mine-active.png` (81x81px) - 选中状态

## 图标规范

- **尺寸**: 81x81 像素
- **格式**: PNG（透明背景）
- **风格**: 简洁线性或面性图标
- **颜色**: 
  - 默认状态：#999999
  - 选中状态：#007AFF

## 临时方案

如果暂时没有图标，可以：
1. 使用 iconfont 等在线图标库下载
2. 使用设计工具（Figma、Sketch）自己绘制
3. 暂时使用文字标签代替图标
