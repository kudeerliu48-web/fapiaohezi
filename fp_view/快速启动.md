# UniApp 移动端应用 - 快速启动指南

## ✅ 已完成的内容

### 📱 三个核心页面

1. **首页 (pages/index/index.vue)**
   - 渐变色头部设计
   - 拍照/相册上传功能
   - 数据统计卡片
   - 快捷操作按钮

2. **历史记录 (pages/history/history.vue)**
   - 搜索栏
   - 发票列表展示
   - 状态标签（待识别/已识别/失败）
   - 下拉刷新、上拉加载
   - 删除和重新识别功能

3. **个人中心 (pages/mine/mine.vue)**
   - 用户信息卡片
   - 功能菜单列表
   - 数据统计、发票导出
   - 批量识别、设置等

### 🔧 配套文件

- `manifest.json` - 应用配置
- `pages.json` - 页面路由和 TabBar
- `App.vue` - 应用入口
- `main.js` - 主入口
- `uni.scss` - 全局样式变量
- `utils/api.js` - API 接口封装
- `pages/login/login.vue` - 登录页面

## 🚀 启动方式

### 方式一：HBuilderX（最简单）

1. **下载 HBuilderX**
   - 访问：https://www.dcloud.io/hbuilderx.html
   - 下载并安装

2. **导入项目**
   - 打开 HBuilderX
   - 文件 > 打开目录 > 选择 `fp_view` 文件夹

3. **运行到浏览器**
   - 点击工具栏：运行 > 运行到浏览器 > Chrome
   - 或使用快捷键：Ctrl + R

4. **查看效果**
   - 访问：http://localhost:8080
   - 可以看到移动端界面

### 方式二：命令行（需要 Node.js）

```bash
# 1. 进入项目目录
cd fp_view

# 2. 安装依赖（如果还没有安装）
npm install

# 3. 运行到 H5
npm run dev:h5

# 4. 打开浏览器访问
# http://localhost:8080
```

## 📋 配置步骤

### 1. 准备 TabBar 图标

在 `src/static/tabbar/` 目录下放置 6 个图标：
- home.png / home-active.png
- history.png / history-active.png
- mine.png / mine-active.png

**临时方案**：如果没有图标，可以暂时去掉 pages.json 中的 iconPath 配置。

### 2. 配置后端 API

编辑 `src/utils/api.js`：

```javascript
const BASE_URL = 'http://localhost:8000/api'
```

确保后端服务已启动：

## 🐛 可能遇到的问题

### 问题 1：TabBar 图标不显示
**解决**：
- 检查图标文件是否存在
- 确认路径正确
- 或暂时移除 iconPath 配置

### 问题 2：API 请求失败
**解决**：
- 确认后端服务已启动（端口 8000）
- 检查 BASE_URL 配置
- 浏览器按 F12 查看网络请求

### 问题 3：样式显示异常
**解决**：
- 清除浏览器缓存
- 重新编译项目
- 检查 SCSS 语法

## 📱 多端运行

### 微信小程序

1. 在 HBuilderX 中：
   - 运行 > 运行到小程序模拟器 > 微信开发者工具
   
2. 配置 AppID：
   - 修改 manifest.json
   - 填入你的小程序 AppID

### App（Android/iOS）

1. 云打包：
   - 发行 > 原生 App-云打包
   
2. 自定义基座：
   - 运行 > 运行到手机或模拟器
