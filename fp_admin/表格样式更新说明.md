# 表格网格式样式更新说明

## 更新内容

已为用户管理和发票管理页面的表格添加了网格式样式，使数据展示更加清晰明了。

## 更新的文件

### 1. Users.vue (用户管理页面)
- ✅ 添加了 `border` 属性到 el-table 组件
- ✅ 增强了表格边框样式
- ✅ 表头背景色设置为浅灰色 (#F5F7FA)
- ✅ 鼠标悬停行高亮效果 (#ecf5ff)
- ✅ 文本溢出省略号显示

### 2. Invoices.vue (发票管理页面)
- ✅ 添加了 `border` 属性到 el-table 组件
- ✅ 增强了表格边框样式
- ✅ 表头背景色设置为浅灰色 (#F5F7FA)
- ✅ 鼠标悬停行高亮效果 (#ecf5ff)
- ✅ 紧凑的单元格间距

### 3. global.scss (全局样式)
- ✅ 添加了表格网格样式增强的全局配置
- ✅ 统一所有 Element UI 表格的网格显示效果

## 样式特性

### 网格边框
```scss
th.el-table__cell, td.el-table__cell {
  border: 1px solid #EBEEF5;  // 明显的网格边框
}
```

### 表头样式
```scss
th.el-table__cell {
  background-color: #F5F7FA;  // 浅灰色背景
  color: #606266;
  font-weight: 600;           // 加粗字体
}
```

### 交互效果
```scss
.el-table__body tr:hover > td {
  background-color: #ecf5ff;  // 鼠标悬停淡蓝色高亮
}
```

## 视觉效果

更新后的表格具有以下特点：

1. **清晰的网格线** - 每个单元格都有明显的边框分隔
2. **突出的表头** - 浅灰色背景 + 加粗字体
3. **良好的交互** - 鼠标悬停时行高亮
4. **紧凑的布局** - 适合展示大量数据
5. **文本省略** - 长文本自动显示省略号

## 技术实现

使用了 Element UI 的 `border` 属性和自定义 SCSS 样式结合的方式：

- `border` 属性：启用 Element UI 自带的表格边框
- 自定义样式：增强边框颜色和表头效果
- `::v-deep` 深度选择器：穿透组件样式
